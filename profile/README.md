# Hootsuite Gaming Community Templates

Community-contributed content calendars, report templates, audit sheets and scheduling presets optimized for gaming creators and streamers in **Hootsuite**.

> These are user-made templates only.  
> Always use the **official** Hootsuite from https://www.hootsuite.com

## What's inside this repository

- 📅 Gaming content calendars (.xlsx / .csv / .json templates)
- 📊 Analytics & performance report templates
- 🔍 Social media audit checklists & worksheets
- 🗓️ Scheduling queue presets (best times, post ideas for gaming niches)
- 📸 Example screenshots and usage notes

All files are plain text / spreadsheets / configs — **no executables, no installers, no modifications** to Hootsuite itself.

## How to use

1. Download the latest release from the **Releases** tab on the right → choose the .zip you need  
2. Extract the archive to any convenient folder  
3. Open official **Hootsuite** (log in at https://www.hootsuite.com if needed)  
4. For calendars/reports: Import .xlsx/.csv into Hootsuite Planner or external tools (Google Sheets/Excel) → copy-paste into Composer or Bulk Scheduler  
5. For presets: Manually apply scheduling queues or best-time recommendations in Hootsuite dashboard  
6. Customize for your channels (Twitch, YouTube, TikTok, X, etc.) and test posts  

**Tip**: Use Hootsuite's built-in "Best Time to Post" to refine these templates further. Backup your current planner before importing new data!
